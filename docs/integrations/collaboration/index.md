# Collaboration Tools

Team communication and collaboration platform integrations.

## Available Integrations

- **[Slack](slack.md)** - Team communication and notifications

## Common Features

Collaboration tool integrations typically provide:
- Message sending and receiving
- Channel/workspace management
- File sharing capabilities
- User and team management
- Real-time notifications

## Authentication

Most collaboration tools use OAuth2 for authentication, requiring:
- App registration with the service
- OAuth scopes configuration
- Token management and refresh
