# E-commerce Extensions

WooCommerce extension integrations.

## Available Integrations

- **[Wholesale Suite](wholesale-suite.md)** - B2B wholesale functionality
- **[Advanced Coupons](advanced-coupons.md)** - Advanced coupon management

## Common Features

E-commerce extensions typically:
- Extend WooCommerce functionality
- Add specialized features (wholesale, coupons, etc.)
- Hook into WooCommerce events
- Provide additional tools for store management
