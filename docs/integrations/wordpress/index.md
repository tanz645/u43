# WordPress Integrations

WordPress core and plugin integrations for the Agentic Workflow plugin.

## Available Integrations

- **[WordPress Core](wordpress.md)** - Hook-based triggers, event system, and filter integration
- **[WooCommerce](woocommerce.md)** - E-commerce integration with order management
- **[WPForms](wpforms.md)** - Form submission and entry management
- **[Contact Form 7](contact-form-7.md)** - Form integration with Flamingo support

## Common Patterns

All WordPress plugin integrations follow similar patterns:
- Hook into WordPress actions/filters
- Use plugin-specific APIs
- Store data in WordPress database
- Leverage WordPress user and permission systems
